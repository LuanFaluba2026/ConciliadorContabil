# Conciliador Contábil
Projeto desenvolvido para automatizar a geração da conciliação contábil a partir do razão mensal de movimentações, trazendo mais agilidade no processo.
### Tecnologias utilizadas:
- Banco de dados SQLite
- Windows Forms - .NET
- ClosedXML para leitura e geração de planilhas.
### Funcionalidades: 
- Cadastro e gerenciamento de empresas com banco de dados.
- Importação de movimentos contábeis utilizando planilha do razão.
- Substituição em lote de movimentações já importadas.
- Gerenciamento de contas contábeis (Adicionar, remover e consultar)
- Gerenciamento completo das movimentações (Adicionar, remover, alterar, encerrar e consultar)
- Importação de saldos anteriores para processamento automático
- Ajuste de prefixos dos históricos da movimentação.
### Modo de Uso:
1. Cadastrar e selecionar empresa desejada
2. Importar razão utilizando a devida competência
3. Conferir planilha final e realizar os ajustes necessários via gerenciamentos.
